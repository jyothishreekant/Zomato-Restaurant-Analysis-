# 🍽️ Zomato Restaurant Analysis

> End-to-End Data Analytics Project using **Excel, SQL, Power BI, and Tableau**

# 📌 Project Overview

This project analyzes the Zomato Restaurant dataset to uncover valuable business insights using modern Data Analytics tools.

The project demonstrates the complete analytics workflow, including:

- Data Cleaning
- Data Transformation
- SQL Analysis
- Data Modeling
- Dashboard Development
- Business Insight Generation

The dashboards help stakeholders understand restaurant performance, customer preferences, pricing trends, ratings, online delivery services, and table booking availability across multiple countries and cities.

---

# 🎯 Business Objectives

- Clean and preprocess raw restaurant data.
- Analyze restaurant performance.
- Compare cities and countries.
- Evaluate ratings and pricing trends.
- Study online delivery and table booking services.
- Build interactive dashboards for decision-making.

# 💡 Key Business Insights

- India has the highest number of restaurants.
- New Delhi has the largest restaurant concentration.
- Restaurant ratings vary across different cities.
- Online delivery services are more popular than table bookings.
- Average cost for two differs significantly across countries.
- Customer preferences vary based on cuisine and location.
